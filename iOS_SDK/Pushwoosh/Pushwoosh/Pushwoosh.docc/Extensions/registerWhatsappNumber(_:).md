# ``Pushwoosh/registerWhatsappNumber(_:)``

@Metadata {
    @DocumentationExtension(mergeBehavior: override)

Registers a WhatsApp phone number for multi-channel messaging.

## Discussion

Associates a WhatsApp-enabled phone number with the current device to enable WhatsApp-based campaigns alongside push notifications. This enables multi-channel communication with users.

The phone number should be in international format.

## Parameters

- number: The user's WhatsApp phone number
