# ``Pushwoosh/registerSmsNumber(_:)``

@Metadata {
    @DocumentationExtension(mergeBehavior: override)
}

Registers an SMS phone number for multi-channel messaging.

## Discussion

Associates an SMS-capable phone number with the current device to enable SMS-based campaigns alongside push notifications. This enables multi-channel communication with users.

The phone number should be in international format.

## Parameters

- number: The user's phone number for SMS
