@Metadata {

The Pushwoosh Application ID.

## Discussion

This property contains your Pushwoosh Application Code (App ID) that identifies your app in the Pushwoosh system. The value is usually retrieved automatically from Info.plist parameter `Pushwoosh_APPID`.

This is a read-only property set during SDK initialization.
